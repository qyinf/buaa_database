# 2023数据库大作业

北航三餐生成系统（`Buaa Club Center`）

前端：vue

后端：django

相关的版本号标注在`requirements.txt`中

主要的功能有浏览北航的食堂和周边的外卖数据，并且可以根据个人BMI动态生成食谱
